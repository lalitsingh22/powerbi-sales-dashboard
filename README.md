# 📊 Sales Data Analysis Dashboard – Power BI

## 📌 Project Overview

This project presents an **interactive Sales Analysis Dashboard built using Microsoft Power BI** to visualize and analyze business sales performance.

The dashboard transforms raw transactional sales data into **clear, interactive visual insights** that help stakeholders monitor key metrics, identify trends, and support data-driven decision making.

The project demonstrates core **data analyst skills**, including data preparation, exploratory data analysis, and interactive dashboard development.

---

# 🎯 Objective of the Analysis

The goal of this project is to analyze sales performance and answer key business questions such as:

* How are sales performing over time?
* Which **products or categories generate the most revenue**?
* Which **regions contribute the highest sales**?
* How do different **sales representatives perform**?

The dashboard helps businesses **monitor KPIs and identify opportunities for growth**.

---

# 📁 Dataset Description

## Raw Dataset

The raw dataset contains **original transactional sales data** before any preprocessing.

It includes information such as:

* Order ID
* Order Date
* Product / Category
* Sales Amount
* Customer / Region
* Sales Person

The raw data may contain **missing values, duplicates, and inconsistent formatting**.

---

## Cleaned Dataset

The cleaned dataset is the **processed version of the raw data** prepared for analysis.

Cleaning and preprocessing ensured:

* Consistent formatting
* Accurate calculations
* Structured data ready for Power BI visualization

This dataset was imported into **Power BI for dashboard development**.

---

# 🧹 Data Cleaning & Preparation

Before loading the data into Power BI, several preprocessing steps were performed:

### Handling Missing Values

* Checked for incomplete records
* Handled or removed missing entries where necessary

### Removing Duplicates

* Identified and removed duplicate records to maintain data accuracy

### Data Formatting

* Standardized **date formats**
* Ensured numerical columns (sales values, quantities) were correctly formatted
* Corrected inconsistent text values

### Data Transformation

* Organized columns for analysis
* Created calculated fields where needed
* Structured the dataset for effective dashboard visualization

---

# 📊 Exploratory Data Analysis (EDA)

Exploratory analysis was performed to understand patterns and trends within the data before building the dashboard.

### Sales Trends

* Analysis of **monthly and yearly sales trends**
* Identification of peak sales periods

### Product / Category Performance

* Identification of **top performing products**
* Contribution of each product category to total revenue

### Regional Sales Performance

* Comparison of sales across **different regions**

### Customer Insights

* Understanding customer purchasing patterns
* Identifying segments contributing to higher revenue

# 📈 Power BI Dashboard

An **interactive Power BI dashboard** was developed to present insights in a visually engaging format.

## Key KPIs

* Total Sales Revenue
* Total Orders
* Total Units Sold
* Top Performing Product
* Best Performing Region

## Charts & Visuals Used

* 📊 Bar Charts – Product / Category Performance
* 📈 Line Charts – Sales Trends Over Time
* 🥧 Pie Charts – Sales Distribution by Region
* 📉 KPI Cards – Key Metrics

## Filters & Slicers

* Region Filter
* Product Category Filter
* Date Filter

These interactive elements allow users to **explore the data dynamically**.

# 🔍 Key Insights

Important insights discovered from the dashboard include:

* A small number of products contribute significantly to overall sales.
* Certain regions consistently generate **higher revenue than others**.
* Sales trends reveal **periods of higher demand** during specific months.
* Performance differences between **sales representatives** highlight opportunities for improvement.

# 💡 Business Recommendations

Based on the analysis, the following actions are recommended:

* Focus marketing strategies on **top-performing products**.
* Implement targeted campaigns in **lower-performing regions**.
* Use historical sales trends for **inventory planning and demand forecasting**.
